# Pattern: Enhanced Delegation Discipline During API Interruptions

**DEFINITION**: A critical orchestration pattern that mandates strict adherence to delegation protocols even during external technical disruptions such as API stream interruptions or connection issues. This ensures behavioral consistency and systemic stability under stress, counteracting the tendency for protocol deviations in adverse conditions.

**CORE INSIGHT**: System disruptions are a test of operational discipline. This pattern establishes that maintaining the "conductor, not performer" role is absolutely paramount during connectivity problems, requiring *stronger* delegation discipline rather than relaxed boundaries. It transforms potential points of failure into opportunities to reinforce robust, resilient workflows.
